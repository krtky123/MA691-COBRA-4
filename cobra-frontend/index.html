Customer Segmentation in Bank Credit Risk Datasets using COBRA 
Bank credit risk dataset analysis helps us make predictions regarding a number of paramateres such as classifying customers as good and bad, accepting or rejecting credit card applications, etc. In this project we use combined regression alternative (COBRA), to combine several machines (models) and predict results. The accuracy is seen to improve as a result of using COBRA.
It is important to note that the German Credit Dataset analysis has a web application corresponding to it, but the Australian dataset analysis does not have one. This is because the latter has modified attribute and value names to meaningless symbols for maintaining data confidentiality
German Credit Dataset analysis

This dataset classifies people described by a set of attributes as good or bad credit risks. The dataset can be found here on UCI. The source and the names for attributes and values are intact.

Australian Credit Dataset analysis

This dataset concerns credit card applications. The dataset can be found here on UCI. The source and names of attributes and values are concealed to protect data confidentiality.
